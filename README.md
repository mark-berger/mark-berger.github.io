# maberger.com

Personal website for [https://maberger.com/](https://maberger.com/) built with [Jekyll](https://jekyllrb.com/) and hosted on [GitHub Pages](https://pages.github.com/).

## Requirements

```sh
bundle install
```

## Usage

```sh
bundle exec jekyll serve --host=0.0.0.0
```
